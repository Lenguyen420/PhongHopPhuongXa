# PhongHopPhuongXa

Frontend React + Vite cho he thong phong hop phuong/xa.

## Yeu cau

- Node.js 20 tro len
- npm 10 tro len

## Cai dat

```bash
npm install
```

Tao file `.env` tu file mau:

```bash
cp .env.example .env
```

## Lenh phat trien

```bash
npm run dev
npm run build
npm run preview
npm run lint
npm run lint:fix
npm run format
npm run format:check
```

## Cau hinh chinh

- Vite dung React plugin va Tailwind CSS plugin.
- Alias `@/*` tro den `src/*`.
- Oxlint kiem tra nhanh code trong `src` va `vite.config.js`.
- Prettier dinh dang code thong nhat.
